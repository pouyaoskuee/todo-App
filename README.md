# Todo App

A simple Todo application built with JavaScript that allows users to create, complete, edit and delete tasks.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- edit tasks with modal
- Save tasks in Local Storage
- Simple and responsive UI
- Filter completed and active tasks

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
